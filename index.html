My Healthy European Food Taste Tracker

Open your browser’s Developer Tools → Console to see the output from script.js.

What is this?

A JavaScript assignment for my Web Development class focused on healthy, mostly European-made favorite foods. It demonstrates variables, data types, objects, arrays (CRUD), loops, conditionals, and functions.

How to view results

  1. Open this page (index.html) in your browser.
  2. Open Developer Tools → Console.
  3. Read logs for each labeled section and problem.
